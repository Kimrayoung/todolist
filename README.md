# todolist

### 사용방법
입력 칸에 할일을 입력한 후 입력버튼을 누르거나 enter을 누르면 입력 칸 아래에 할일이 추가됨
할일은 checkbox로 되어있어서 할일을 다한 후 체크박스를 누르면 취소선이 생김
